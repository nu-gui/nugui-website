Benefits of This Approach

  • ‚úÖ Single source of truth - All image paths in one CSS file
  • ‚úÖ Easy to maintain - Change images in one place
  • ‚úÖ Better performance - CSS is cached, no inline styles
  • ‚úÖ Automatic theme switching - Uses native CSS media queries

How to Add New Images

  1. Add image files: new-feature-1.png (light) and new-feature-2.png (dark)
  2. Update /css/theme-assets.css: 
    :root {
    --icon-new-feature: url('/assets/images/new-feature-1.png');
}

@media (prefers-color-scheme: dark) {
    :root {
        --icon-new-feature: url('/assets/images/new-feature-2.png');
    }
}
  3. Use in HTML: <div class="bg-icon-new-feature"></div>
